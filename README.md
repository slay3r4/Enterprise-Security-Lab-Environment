# Enterprise-Security-Lab-Environment
Architected a fully virtualized enterprise security lab using VirtualBox with Kali Linux, Windows Server/Client, and pfSense. Implemented VLAN segmentation, Active Directory, SIEM (Splunk, ELK), IDS/IPS (Suricata), and conducted penetration testing with Metasploit and Burp Suite, automating analysis with Python and Bash.
